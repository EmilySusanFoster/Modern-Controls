Part V: Advanced SISO Control

PREVIEW

In this fifth part of the book we move onto some slightly more advanced topics in SISO control. In particular, we show how to parameterize all linear time invariant controllers that stabilize a given linear system. We then use this parameterization to introduce optimization based methods for control system design. We then turn to alternative methods for control system design based on state space models. These methods provide us with a smooth transition to multivariable control systems which will be the subject of the remaining parts of the book. Finally we introduce some basic ideas related to the control of nonlinear systems.
