Key features of this kind of control problem:
 

  • P.Q.A.’s not well defined nor directly measurable
    (Need nonlinear soft sensors)
     

  • Limited Control authority
    (Need to respect control constraints)
     

  • Strict Limits on certain output variables
    (Need to respect state constraints)

  
 

G. C. Goodwin - Bode Lecture 1999
