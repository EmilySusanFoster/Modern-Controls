A Note About Web Browsers

I have personally tested the Java applets on this site with the following browsers:

  • Netscape Communicator 4.05 (Unix)
  • Netscape Communicator 4.08 (Windows)
  • Netscape Communicator 4.5 (Windows)
  • Netscape Communicator 4.61 (Windows)
  • Internet Explorer 3.0 (Windows)
  • Internet Explorer 4.01 (Windows)
  • Internet Explorer 5.0 (Windows)

I assume that the site will work correctly with other Netscape 4.0x browsers. I have heard that the site works fine with:

  • Netscape Navigator 2.02 (Windows).

Great effort has been made to ensure compatibility with as many browsers as possible, but some browser bugs just can't be worked around:

Netscape 4.08 (Unix) This version of Netscape seems to crash quite regularly when running Java applets (try pressing reload when a Java applet is on the screen), and not just the Java applets on this page. (A Java applet should never be able to crash the browser)
Netscape 3.04 (Windows) The Java implementation in this browser seems a little "strange", with various functions missing from the API! Some of the applets seem to work fine, but others don't and can even cause Netscape to crash

However, if you do run the applets with these browsers and have no troubles, then I'd like to hear about it - send me an email (nathan@ee.newcastle.edu.au) telling me about your system and the exact version of your browser.

If you are running LYNX, then I really do pity you, but there's not much I can do for you in the way of interactive demos. You should still be able to view most of the content, however most of the equations used on the site are graphic images.
