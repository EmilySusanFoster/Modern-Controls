Netscape Navigator

To get the latest version of Nescape Navigator from Netscape's website, click here.

    

To install Netscape Navigator version 4.73 directly from the CD, follow these directions:

  1. Open Windows Explorer.
  2. Click on the CD drive that has this CD in it.
  3. Open the Netscape folder.
  4. Double click on the net4_73.exe file to run the installer.

These pages are optimised for Netscape and Internet Explorer browsers versions 4 and above, and the pages are designed so that they can be viewed at 640x480 resolution, but they are best viewed at a resolution of 800x600 or greater.

IMPORTANT: Read this note about browsers if you are using a version 3 browser.
